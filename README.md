# Train Information System

Console application written in C# for working with text files.

## Description

The program creates a file named `INFO.TXT` that stores information about trains departing from Khmelnytskyi station.

Each train contains:
- train number
- destination station
- departure time
- travel time

After saving data to the file, the program reads the file and displays trains that depart no later than 18:00.

---

## Features

- Writing data to a text file
- Reading data from a text file
- Working with `StreamWriter`
- Working with `StreamReader`
- Using classes and methods
- Time comparison using `TimeSpan`
